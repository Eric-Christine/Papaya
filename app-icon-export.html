iOS App Icon Generator

📱 How to use this icon in your Papaya app:

  1. Click the "Download 1024x1024 PNG" button above
  2. Save the file as icon.png
  3. Place it in assets/images/icon.png in your Papaya project
  4. Update app.json:
    "icon": "./assets/images/icon.png"
  5. Rebuild your app:
    npx expo prebuild --clean
    npm run ios or npm run android
